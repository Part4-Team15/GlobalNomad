# 🌎 Global Nomad

<img src="https://github.com/Part4-Team15/GlobalNomad/assets/90647684/0a00f90a-e7f0-4503-a4a9-7b71f27c1862" width="100%"/>

```
사람들이 여행을 갈 때, 가서 뭘 할지, 비용은 얼마인지 등 여러 고민을 하게 된다.
바쁜 현대인의 이런 고민을 줄여주기 위해 플랫폼 안에 잘 짜인 체험 상품을 보고 간단하게 예약할 수 있는 서비스입니다.
```

<br/>

## 🔗 배포 링크

> https://global-nomad-sprint4-15.vercel.app/

<br/>

## 🗓️ 개발 기간

> 2024.05.20 ~ 2024.06.25

<br/>

## 목표

- React Query를 활용한 비동기 상태 관리 이해
- 지도, 캘린더 API를 활용한 구현
- 페이지별 컴포넌트 작업을 통한 협업
- Tailwind CSS의 이해 및 적용
